# 鼠标手写签名生成器

[English](README.md)

这是一个本地桌面工具：将中文、英文或中英文混合姓名转换为笔迹轨迹，再在用户框选的屏幕区域内，通过真实的鼠标按下、移动和松开事件完成绘制。

它适用于本地应用、PDF 阅读器、绘图画布和网页签字控件的签字流程测试。程序不会上传屏幕内容、键盘输入或签名数据。

## 功能

- 支持中文、英文及混合姓名。
- 真实回放前可预览签名字迹。
- 用户框选签字区域；回放时会再次裁剪坐标，避免轨迹越界。
- 可调整大小、速度、随机扰动和倾斜角度。
- 根据局部曲率生成逐点速度变化。
- 回放前提供 3 秒倒计时。
- 使用 `pynput` 支持全局 ESC 取消；同时保留 PyAutoGUI 鼠标移至屏幕角落的故障保护。
- 支持 macOS Quartz、Windows Win32 和 PyAutoGUI 后备鼠标控制。

## 环境要求

- Python 3.12+
- [uv](https://docs.astral.sh/uv/)
- 需要实际回放鼠标时，使用 macOS 或 Windows。

## 安装与运行

```bash
cd MouseSign
uv sync
./run.sh
```

也可执行 `uv run python main.py`。

## 使用方法

1. 输入姓名。
2. 调整大小、速度、扰动和倾斜参数。
3. 点击“预览”。
4. 点击“选择区域”，拖拽框选唯一允许接收鼠标事件的区域。
5. 点击“开始签名”，并在倒计时内切换到目标应用。
6. 任意时刻按 ESC 可停止。若全局输入监听不可用，将鼠标移到主显示器任一角落以触发 PyAutoGUI 故障保护。

## 字库与当前范围

- 英文使用仓库内置的 Hershey 单线字体数据：`fonts/hersheytext.json`。
- 中文使用 [animCJK](https://github.com/parsimonhi/animCJK) 的标准笔顺中心线。程序优先读取 `fonts/svgsZhHans` 中的缓存 SVG；未缓存汉字会首次联网下载并写入缓存。
- animCJK 是标准笔顺数据，不是真人在线手写样本。当前中文输出可辨识、可按笔顺回放，但不是同一书写者风格的真实签名。
- CASIA 等真实在线手写数据不包含在项目中。若要实现真实中文签名风格，必须自行合法获取数据集并接入对应笔迹样本。

请先在系统画图工具等可丢弃画布中验证，再用于真实签字流程。

## macOS 权限

在“系统设置 → 隐私与安全性”中，为终端或 Python 应用授予：

- **辅助功能**：发送鼠标事件。
- **输入监控**：在其他应用获得焦点时监听全局 ESC。

## 开发验证

```bash
uv run python -m unittest discover -s tests -v
```

## 项目结构

```text
core/       轨迹生成、文字转换、鼠标回放、紧急停止
fonts/      Hershey 字体数据和缓存的中文 SVG 轨迹
gui/        Tkinter 主界面与区域框选覆盖层
tests/      可重复执行的核心回归测试
```
